Getting Started

The design system and styleguide behind IT products at CareerBuilder.

Welcome to Kabu, an internal guide & code repository for designing apps at CareerBuilder.

Kabuv1.0.0
Installation

To get going with Kabu, simply download our default CSS package.

Download Kabu

Setting up your project

After downloading the CSS package, unzip and move the folder to your project directory. The template below has the minimum boilerplate to get started.

Don't Repeat Yourself

Be sure to check out our prebuilt templates to see if there's already a page that suits your needs.

Copy 
<!DOCTYPE html>
<html>
<head>
  <meta charset='UTF-8'/>
  <meta name='viewport' content='initial-scale=1,maximum-scale=1,user-scalable=no' />
  <title>CareerBuilder</title>
  <!-- Our example uses kabu as the folder name. -->
  <link href='kabu/css/kabu.min.css' rel='stylesheet' />
</head>
<body>
  <!-- Content goes here -->
  <script src='https://ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js' ></script>
  <script src='kabu/js/kabu.min.js'></script>  
</body>
</html>
