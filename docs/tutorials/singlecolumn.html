In this example let's create a single column to-do list.

1. Create a container

Start by creating a container for our to-do list. Let's use the .box class and center it to the grid with an *-offset class. .round takes care of rounding its corners and softening the edges.

container
Copy 
<div class='box round col-sm-10 col-sm-offset-1'>
  container
</div>

2. Add our to-do items

To-do items need space to breathe and a bottom border to separate them from other items in the list. To accomplish this we'll use the .padx, .pady and .keyline-bottom classes. Learn more about padding and keyline helpers by visiting our grid and helpers sections.

Catch all 648 Pokemon
Save the cheerleader
Clone dinosaurs
Destroy 7 horcruxes
Get back to 1985
Copy 
<div class='box round col-sm-10 col-sm-offset-1'>
  <div class='pad1y pad2x keyline-bottom'>Catch all 648 Pokemon</div>
  <div class='pad1y pad2x keyline-bottom'>Save the cheerleader</div>
  <div class='pad1y pad2x keyline-bottom'>Clone dinosaurs</div>
  <div class='pad1y pad2x keyline-bottom'>Destroy 7 horcruxes</div>
  <div class='pad1y pad2x'>Get back to 1985</div>
</div>

3. Include some icons

We want to easily distinguish which items in our list have been completed. To do that let's use the .icon-ok icon.

Let's line up our to-do items using the grid system. We'll make the icon one column with .col-xs-1. Finally we'll add a .clearfix class to clear floats on each item.
