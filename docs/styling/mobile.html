Responsive utitlies

Mobile layout is handled automatically by the grid system using the *-lg, *-md, *-sm, and *-xs classes. Also included are utility classes that can help with toggling content across viewport breakpoints. For a complete list see Bootstrap's documentation on responsive utilities.
