Typeface

Open Sans is our default typeface for websites, print and other materials. Designed with open forms and a friendly appearance, Open Sans carefully crafted glyphs and clean, modern lines also fit our Flat UI style.

Headings

Use <h1> through <h4> to add visual hierarchy to your page. <h5> and <h6> are given the same styling and should only be used to add semantic structure.

Create secondary headings or subtitles by adding .small and .quiet classes to the heading.

Links

Links are styled globally using the styleguide's primary color. Customize this variable and others within variables.less.

Body copy

Our default font-size is 15px with a line-height of 20px. This is applied to the <body> and all paragraphs. As expected the <p> tag receives a bottom margin to create space.

Dark themed text

To add white text on a dark background combine the dark class with a fill-* from the colors palette on a containing element.

Light Text on Dark Backgrounds

Use caution when choosing dark themed text. Studies have shown that having light text on a dark background harms readibility and is generally a bad idea.
