Progress Spinner

Use a progress spinner to provide feedback to users while a process is running. Variations are available based on classes applied to the circle. Add class .dark for a gray spinner or class .loud for a colorful spinner.
A .largeclass for the default spinner is also available.

.path
.path .loud
.path .dark
Copy 
<div class='loader'>
  <svg class='circular'>
    <circle class='path' cx='50' cy='50' r='10' fill='none' stroke-width='3' stroke-miterlimit='10'/>
  </svg>
</div>
.path .large
Copy 
<div class='loader'>
  <svg class='circular'>
    <circle class='path large' cx='50' cy='50' r='20' fill='none' stroke-width='3' stroke-miterlimit='10'/>
  </svg>
</div>
