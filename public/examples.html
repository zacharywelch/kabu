Blank Slate

Blank slate is the screen people see when they start using an application and there is no data to show yet. Leaving it blank would not make a good first impression, so it should be filled with something useful to make the impression positive and help people get started.

The blank slate block contains a large add icon, heading and paragraph for instructions. You can modify any element of this block to fit your needs, but you should follow the same HTML structure to pick up its styling.
