Getting Started

1

It's time to get started

Getting started with CareerBuilder design framework is easy. Drop the desired example into your project and base your design on it. It's that simple!

Variety of designs

You can decide whether to create your site using basic building blocks or complete examples. Combine blocks in an endless variety of design projects. It's easy!

2

Design in the browser

Patterns were built in code to make creating a website quick and easy. This setup will allow you to swiftly prototype your ideas and start validating them sooner rather than later.

Based on Twitter Bootstrap and Flat UI

HTML layout is based on today's most popular framework, Bootstrap 3. Bootstrap components are then given a stunning flat-style using the Flat UI toolkit.

Build across devices

Create responsive sites and apps that work on any device with a library of tested components. Our layout works perfectly on any modern platform, including mobile. No extra code required.

3

Stay current

Our goal is to create compelling user experiences. We are interested in further developing CareerBuilder design framework to meet your project's needs. As we improve existing components we'll also be adding new ones that can be reused across your projects.
